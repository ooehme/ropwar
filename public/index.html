XR-Anker

Diese Version hat keinen Kamera-/Sensor-Fallback. Das Windrad wird nur mit WebXR immersive-ar gerendert. Die Zusatzknöpfe dienen zur Sichtprüfung und zur manuellen Korrektur der Geo-Ausrichtung. Die WebXR Depth API wird jetzt immer als Pflichtfunktion beim AR-Start angefordert; ohne Depth startet die AR-Session nicht.
